public interface Func0Async<R>

A Func0Async is a reference to a function with return value that can be invoked asynchronously in another thread. With Func0Async it is possible to track the status and outcome using the FuncAsyncResult returned upon invocation or by defining an FuncAsyncCallback for the invocation.

beginInvoke

FuncAsyncResult<R> beginInvoke(FuncAsyncCallback<? super R> callback,
                               Object asyncState)
Asynchronously invoke this Func0Async with the given parameter value, callback and asyncState. Associated with each call to beginInvoke is a FuncAsyncCallback instance returned by the call and given to the callback that can be used to track the status and outcome of the invocation.

Parameters:
callback - the FuncAsyncCallback, if given to beginInvoke, is invoked upon completion of the Func0Async's invocation and receives the same FuncAsyncResult that is returned by the call to beginInvoke.
asyncState - the asyncState is an arbitrary reference that, if given to beginInvoke, can be retrieved from this invocations FuncAsyncResult.getAsyncState()
Returns:
the FuncAsyncResult associated with this asynchronous invocation

public class DynamicFuncToDynamicFuncAsync
extends Object
implements DynamicFuncAsync

An adapter that converts a conventional DynamicFunc to an DynamicFuncAsync. All asynchronous calls are executed in another thread and forwarded to the DynamicFunc.dynamicInvoke(Object...) method. The Executor to use for the asynchronous invocations can be chosen during creation of this adapter. If no explicit executor is given the a default executor is used. The default executor can be set using the AsyncExecutor.setDefaultExecutor(Executor) property or is created automatically by the AsyncExecutor.

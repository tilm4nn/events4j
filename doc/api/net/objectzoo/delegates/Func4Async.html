public interface Func4Async<T1,T2,T3,T4,R>

A Func4Async is a reference to a function with return value that can be invoked asynchronously in another thread. With Func4Async it is possible to track the status and outcome using the FuncAsyncResult returned upon invocation or by defining an FuncAsyncCallback for the invocation.

Asynchronously invoke this Func4Async with the given parameter value, callback and asyncState. Associated with each call to beginInvoke is a FuncAsyncCallback instance returned by the call and given to the callback that can be used to track the status and outcome of the invocation.

Parameters:
callback - the FuncAsyncCallback, if given to beginInvoke, is invoked upon completion of the Func4Async's invocation and receives the same FuncAsyncResult that is returned by the call to beginInvoke.
asyncState - the asyncState is an arbitrary reference that, if given to beginInvoke, can be retrieved from this invocations FuncAsyncResult.getAsyncState()
parameter1 - the first parameter's value for the invocation
parameter2 - the second parameter's value for the invocation
parameter3 - the third parameter's value for the invocation
parameter4 - the fourth parameter's value for the invocation
Returns:
the FuncAsyncResult associated with this asynchronous invocation

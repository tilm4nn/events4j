public interface FuncAsync<T,R>

A FuncAsync is a reference to a function with return value that can be invoked asynchronously in another thread. With FuncAsync it is possible to track the status and outcome using the FuncAsyncResult returned upon invocation or by defining an FuncAsyncCallback for the invocation.

FuncAsyncResult<R> beginInvoke(FuncAsyncCallback<? super R> callback,
                               Object asyncState,
                               T parameter)
Asynchronously invoke this FuncAsync with the given parameter value, callback and asyncState. Associated with each call to beginInvoke is a FuncAsyncCallback instance returned by the call and given to the callback that can be used to track the status and outcome of the invocation.

Parameters:
callback - the FuncAsyncCallback, if given to beginInvoke, is invoked upon completion of the FuncAsync's invocation and receives the same FuncAsyncResult that is returned by the call to beginInvoke.
asyncState - the asyncState is an arbitrary reference that, if given to beginInvoke, can be retrieved from this invocations FuncAsyncResult.getAsyncState()
parameter - the parameter value for the invocation
Returns:
the FuncAsyncResult associated with this asynchronous invocation

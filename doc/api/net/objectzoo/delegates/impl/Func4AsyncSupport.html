This is an abstract support class that can be used to quickly implement asynchronous funcs. All asynchronous calls are executed in another thread and forwarded to the Func4.invoke(Object, Object, Object, Object) method. The Executor to use for the asynchronous invocations can be chosen during creation of ActionAsyncSupport instances. If no explicit executor is given the a default executor is used. The default executor can be set using the AsyncExecutor.setDefaultExecutor(Executor) property or is created automatically by the AsyncExecutor.

Asynchronously invoke this Func4Async with the given parameter value, callback and asyncState. Associated with each call to beginInvoke is a FuncAsyncCallback instance returned by the call and given to the callback that can be used to track the status and outcome of the invocation.

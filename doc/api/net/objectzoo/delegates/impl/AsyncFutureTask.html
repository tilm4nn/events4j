All Implemented Interfaces:

public class AsyncFutureTask<R>
extends FutureTask<R>
implements FuncAsyncResult<R>, ActionAsyncResult

This FutureTask subclass is used internally for the asynchronous invocation of actions and funcs and implements the interfaces FuncAsyncResult and ActionAsyncResult so that it directly serves as return value for the asynchronous invocations and as parameter given to the callback methods.

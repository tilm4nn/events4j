public abstract class Action2AsyncSupport<T1,T2>
extends Object
implements Action2<T1,T2>, Action2Async<T1,T2>

This is an abstract support class that can be used to quickly implement asynchronous actions. All asynchronous calls are executed in another thread and forwarded to the Action2.invoke(Object, Object) method. The Executor to use for the asynchronous invocations can be chosen during creation of ActionAsyncSupport instances. If no explicit executor is given the a default executor is used. The default executor can be set using the AsyncExecutor.setDefaultExecutor(Executor) property or is created automatically by the AsyncExecutor.

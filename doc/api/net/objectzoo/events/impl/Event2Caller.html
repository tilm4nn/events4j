public class Event2Caller<T1,T2>
extends Object
implements Event2Delegate<T1,T2>

The Event2Caller is a Event2Delegate implementation that allows event distribution to a single subscriber. It is a helper class that encapsulates all the logic required to provide events in other classes. Subscription of excessive subscribers results in IllegalStateExceptions to be thrown.

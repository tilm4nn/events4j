public interface EventAsyncDelegate<T>
extends ActionAsync<T>, EventDelegate<T>

An EventAsyncDelegate implements ActionAsync as well as EventDelegate and is meant as a specification for implementations that delegate calls to the ActionAsync interface asynchronously to all subscribers of the EventDelegate interface.

Class EventDistributor<T>

All Implemented Interfaces:

public class EventDistributor<T>
extends Object
implements EventDelegate<T>

The EventDistributor is a EventDelegate implementation that allows event distribution to multiple subscribers. It is a helper class that encapsulates all the logic required to provide events in other classes. Subscriber invocations are performed in the order of subscription.
